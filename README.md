# Application

## Brew

**install**

    brew update
    brew upgrade
    brew install git
    brew install asdf
    brew install watchman
    brew install git-flow-avh

## ASDF

**install**

    asdf install nodejs 18.20.4

## Npm

**install**

    npm install

## Expo

**commands**

    npx expo start
    npx expo start -c

    npx expo prebuild
    npx expo prebuild --platform ios
    npx expo prebuild --platform android

    npx expo run:android
    npx expo run:ios
